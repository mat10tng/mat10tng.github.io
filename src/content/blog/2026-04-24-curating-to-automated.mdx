## Principle
Automation removes the human bookkeeping required to maintain a self-curating system. The Model Context Protocol (MCP) enables direct, autonomous model interaction with the file system.

## Automation Toolset
- **`read_lessons`**: Retrieves standards autonomously.
- **`save_session`**: Records outcomes directly to project directories.

## Bridge Implementation
The bridge script exposes the file system to the model, allowing it to perform its own retrieval and logging without manual prompting.
</div>

<div slot="technical">
```javascript
// index.js (Simplified)
server.setRequestHandler(CallToolRequestSchema, async (request) => {
  if (request.params.name === "save_session") {
    const { project, content } = request.params.arguments;
    const dir = path.join(VAULT_PATH, project, "sessions");
    await fs.mkdir(dir, { recursive: true });
    await fs.writeFile(path.join(dir, `${Date.now()}.md`), content);
    return { content: [{ type: "text", text: "Logged." }] };
  }
});
```
</div>
