title: "the system that learns"
description: "9 out of 10 work sessions now reuse something a previous one wrote. that wasn't always true."
pubDate: 2026-03-21
loop: "measure"
---

the notes started as storage. write things down, search when needed. most were never found again.

the shift was measuring it. what percentage of work sessions actually reuse something from a previous one vs starting from scratch? first measurement: near zero. every conversation was reinventing.

so I started tracking. every search, every document opened, every note written — all logged. which notes show up in results? which ones actually get read? showing up but never opened = noise. rarely showing up but always read = gold.

then the system started curating itself. notes get written automatically after each session. after enough pile up, they get compressed into lessons. lessons that keep getting referenced get promoted. notes that never get referenced get archived. the useful stuff floats up.

9 out of 10 sessions now pull from something a previous one wrote. the system remembers patterns I've forgotten, surfaces fixes I applied weeks ago, catches me before I repeat a mistake.

the measurement made it real. without the number, "the notes are useful" is just a feeling. with it, it either goes up or it doesn't.
