## Principle
Automation removes the human bookkeeping required to maintain a self-curating system. The Model Context Protocol (MCP) enables direct, autonomous LLM interaction with the Memory.

## Automation Toolset
- **`read_lessons`**: Retrieves standards autonomously from `[root]/memory/`.
- **`save_session`**: Records outcomes directly to project directories.

<div slot="insight">
## Interface Implementation
The interface script exposes the file system (Memory) to the LLM, allowing it to perform its own retrieval and logging without manual prompting.
</div>

<div slot="technical">
```javascript
// index.js (Simplified)
const MEMORY_PATH = path.join(os.homedir(), "memory");

server.setRequestHandler(CallToolRequestSchema, async (request) => {
  if (request.params.name === "save_session") {
    const { project, content } = request.params.arguments;
    const dir = path.join(MEMORY_PATH, project, "sessions");
    await fs.mkdir(dir, { recursive: true });
    await fs.writeFile(path.join(dir, `${Date.now()}.md`), content);
    return { content: [{ type: "text", text: "Logged." }] };
  }
});
```
