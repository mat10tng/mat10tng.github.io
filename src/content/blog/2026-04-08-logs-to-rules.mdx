<Section>
<div slot="insight">
## Principle
Standards are patches for specific project problems that prove to have universal utility.

## Graduation Path
1. **Detection**: Identify recurring friction.
2. **Local Fix**: Implement project-level rule.
3. **Pattern**: Observe fix solving issues in unrelated projects.
4. **Graduation**: Move to global standards.
</div>

<div slot="technical">
| Step | Action |
|---|---|
| **Capture** | Log friction in session file. |
| **Fix** | Add rule to project `lessons/`. |
| **Move** | Relocate to `global/lessons/`. |
</div>
</Section>

<Section>
<div slot="insight">
## Distillation Skill
Automate graduation via periodic review of session logs.
</div>

<div slot="technical">
**Command:** `/distill`
**Logic:** "Review last 7 days of logs. Identify 3 recurring patterns. Move to `global/lessons/`. Update project bootloader."
</div>
</Section>

<Section>
<div slot="insight">
## Result
A self-curating system where high-utility logic inherits automatically, reducing instruction overhead.
</div>
