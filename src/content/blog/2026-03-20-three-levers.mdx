## Abstract
Agent deviation is rarely a code bug. Diagnosis localizes to three structural levers: Instructions, Capability, and Environment. Investigate in that order to resolve behavioral issues.

## 1. The Instructions (Prompt)
If the model is capable but fails to execute, the instruction set is insufficient. **Fix**: Explicit, mandatory rules in System Rules.
</div>

<div slot="technical">
```markdown
- **Rule:** You MUST verify the build.
- **Rule:** Do NOT use library X.
```
</div>
</Section>

<Section>
<div slot="insight">
## 2. The Tools (Capability)
Structural ambiguity caused by redundant tools leads to inconsistent choices. **Fix**: Consolidate into uniquely named functions.
</div>

<div slot="technical">
- Merge overlapping MCP tools.
- Use precise, functional names.
</div>
</Section>

<Section>
<div slot="insight">
## 3. The Limits (Environment)
System limits (timeouts, iteration budgets) prevent task completion. **Fix**: Raise limits or automate bookkeeping.
</div>

<div slot="technical">
```env
MAX_ITERATIONS=15
TIMEOUT_MS=30000
```
</div>
</Section>
