<div slot="insight">
## Principle
Agent deviation is rarely a code bug. Diagnosis localizes to three structural levers.
</div>
</Section>

<Section>
<div slot="insight">
## 1. The Prompt (Instruction)
Insufficient instruction or lack of explicit prohibition. 
**Fix**: Mandatory rules in bootloader.
</div>

<div slot="technical">
```markdown
- **Rule:** You MUST verify build.
- **Rule:** Do NOT use library X.
```
</div>
</Section>

<Section>
<div slot="insight">
## 2. The Tools (Capability)
Structural ambiguity caused by redundant or overlapping tools.
**Fix**: Distinct, uniquely named functions.
</div>

<div slot="technical">
- Merge overlapping tools.
- Rename for precision.
</div>
</Section>

<Section>
<div slot="insight">
## 3. The Constraints (Environment)
System limits preventing completion (e.g., small iteration budgets).
**Fix**: Automate bookkeeping; raise budgets.
</div>

<div slot="technical">
```env
MAX_ITERATIONS=15
```
</div>
</Section>

<Section>
<div slot="insight">
## Order
1. Instruction → 2. Tooling → 3. System Limits.
</div>
</Section>
