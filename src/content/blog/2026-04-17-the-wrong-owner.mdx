<div slot="insight">
## Principle
Reliability is achieved by removing probabilistic tasks from the AI and transferring them to deterministic code.

## The Logic
When a task fails, ask: "Should the model be doing this at all?"
</div>

<div slot="technical">
| Task | Correct Owner |
|---|---|
| **Bookkeeping** | Code/Database |
| **Enforcement** | CI/Scripts |
| **Synthesis** | AI Model |
</div>
</Section>

<Section>
<div slot="insight">
## Example: ID Uniqueness
Instead of instructing the model to "be careful" with IDs, implement a validation function.
</div>

<div slot="technical">
```javascript
function validateId(id, set) {
    if (set.has(id)) throw Error("Duplicate");
}
```
</div>
</Section>

<Section>
<div slot="insight">
## Result
Transferring the job is more effective than refining the instruction.
</div>
