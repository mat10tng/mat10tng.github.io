## Abstract
Probabilistic models are unreliable for deterministic tasks like counting, bookkeeping, or ID management. Reliability is achieved by transferring these responsibilities to the surrounding system logic.

## Task Ownership
When a task fails, verify if it should be delegated to the LLM.
</div>

<div slot="technical">
| Task | Correct Owner |
|---|---|
| **Bookkeeping** | Database / Code |
| **Enforcement** | CI / Validation Scripts |
| **Synthesis** | AI Model |
</div>
</Section>

## Implementation: ID Validation
Instead of instructing a model to "be careful" with identifiers, implement a hard check in the interface tool.
</div>

<div slot="technical">
```javascript
function validateId(id, set) {
    if (set.has(id)) {
        throw Error("Deterministic Error: Duplicate ID");
    }
}
```
