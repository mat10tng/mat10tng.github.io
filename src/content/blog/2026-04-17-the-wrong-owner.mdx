<div slot="insight">
**ELI5:** AI is great at writing but bad at counting things exactly. If the AI keeps making mistakes with numbers or page IDs, stop asking it to "try harder." Instead, write a simple script that handles the counting for it.

## Principle
Fix "hallucinations" by taking the job away from the AI and giving it to a simple bit of code.
</div>

<div slot="technical">
| Task | Best Performer |
|---|---|
| **Counting & Math** | Code / Database |
| **Enforcing Rules** | Scripts / CI |
| **Brainstorming** | AI Model |
</div>
</Section>

<Section>
<div slot="insight">
## Example: Unique IDs
Instead of telling the model to "please don't duplicate IDs," give it a tool that checks for duplicates automatically.
</div>

<div slot="technical">
```javascript
// Simple ID Check
function checkDuplicate(id, list) {
    if (list.includes(id)) {
        throw Error("Duplicate ID found!");
    }
}
```
</div>
</Section>

<Section>
<div slot="insight">
## Result
The system becomes more stable because the AI doesn't have to worry about bookkeeping. It can focus on the creative work.
</div>
